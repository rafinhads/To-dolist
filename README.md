<h1>To-Do List</h1>
A simple and efficient to-do list application built with Java and HTML.

### Features
- Add, edit, and delete tasks
- Mark tasks as completed
- Filter tasks by status (all, active, completed)

### Technologies Used
- Java
- HTML

### Getting Started

**Prerequisites**
- Java Development Kit (JDK)
- Maven
